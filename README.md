# Тайм-менеджер (лично для автора)

**АВТОР:** Николай Матвеев

---

### ОБЩИЕ ТРЕБОВАНИЯ

Программа хранит задачи с различными характеристиками: 

* Дедлайн (если есть)
* Обязательность
* Ориентировочное время на выполнение
* Тип (прога, сложная матеша, простая матеша, бытовая, бюрократическая, хобби, социальная и т.п.)
* Совместимость с другими типами
* Приемлемый диапазон непрерывной работы (некоторые задачи можно адекватно делать только большими непрерывными кусками)
* Степень получения удовольствия от занятия
* И другие

**Общие целевые критерии:**
* Сколько остаётся свободного времени
* Удовлетворение от выполненного
* Важность закрытых и незакрытых задач
* И другие
*(Например: свободного времени не менее двух часов в неделю, максимальная важность незакрытых задач = 6/10, удовлетворение от выполненного устремить к максимуму)*

**Главная цель:**
Возможность создать ориентировочный план для выполнения установленных задач с установленными требованиями (или констатировать невозможность и предложить изменения).

---

### ФИЧИ

#### Первая итерация
1. Ввод и хранение задач и критериев, выведение задач в отсортированном виде по любому критерию.
2. Графический интерфейс (базовый).

#### Вторая итерация
3. Составление плана по критериям (возможно с использованием машинного обучения для поиска экстремумов).
4. Графический интерфейс (удобный).
5. Создание бота в ТГ и/или системы предложения изменений для невозможных планов.
